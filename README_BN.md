# শাহানাজ প্যাথলজি পোর্টাল (shahnazpathology.org) - এন্টারপ্রাইজ অটোমেশন ও ডিপ্লয়মেন্ট গাইড
---

## ১. ডাক্তার ও রোগীদের আস্থা অর্জনে ৮টি সবচেয়ে নির্ভরযোগ্য প্রযুক্তিগত বৈশিষ্ট্য (High-Value Premium Features)

আমাদের ইএইচআর (EHR) সিস্টেমে এমন কিছু বিশ্বমানের আধুনিক প্রযুক্তি যুক্ত করা হয়েছে যা বাংলাদেশের সাধারণ মানুষ, রোগী এবং সম্মানিত চিকিৎসকদের দোরগোড়ায় শতভাগ নির্ভুল, দ্রুত এবং নির্ভরযোগ্য সেবা পৌঁছে দেবে। নিচে চিকিৎসকদের জন্য সিস্টেমেটিক বিবরণ প্রদান করা হলো:

### ক. দ্বিমুখী ল্যাব অ্যানালাইজার মেশিন ইন্টারফেসিং (Bi-directional Lab Interfacing)
আমদানিকৃত টেস্টিং ডিভাইস (যেমন—Roche, Sysmex, Abbott) থেকে কোনো প্রকার মানুষের স্পর্শ ছাড়াই সরাসরি সফটওয়্যারে ডেটা ট্রান্সফার নিশ্চিত করা হয়। এতে ট্রান্সক্রিপশন বা টাইপিং ভুলের আশঙ্কা ০% এ নেমে আসে।

### খ. ক্লাউড ডাইকম ও প্যাক্স ওয়েব ভিউয়ার (DICOM & PACS Cloud Web Viewer)
ল্যাবরেটরীর এক্স-রে, সিটি স্ক্যান বা এমআরআই ইমেজ ডাইকম স্ট্যান্ডার্ডে ক্লাউড সার্ভারে সংরক্ষিত থাকে। চিকিৎসকরা যেকোনো ডিভাইস থেকে রিয়েল-টাইমে হাই-রেজোলিউশন ইমেজ ভিউয়ারের সাহায্যে রিপোর্ট পরীক্ষা করতে পারেন।

### গ. সম্পূর্ণ অফলাইন অপারেশন ও লোকাল হাইব্রিড সার্ভার নোড (Offline Failover Mode)
ইন্টারনেট সংযোগ সাময়িকভাবে বিচ্ছিন্ন থাকলেও ডায়াগনস্টিক কার্যক্রম ব্যাহত হবে না। আমাদের লোকাল রিডান্ডেন্ট হাইব্রিড সার্ভার অফলাইনেও টেস্ট এন্ট্রি, বারকোড স্ক্যানিং এবং প্রেসক্রিপশন জেনারেশন সচল রাখবে এবং ইন্টারনেট ফিরে এলে তা ক্লাউডে সিঙ্ক করে নেবে।

### ঘ. এনক্রিপ্টেড কিউআর-কোড ও ডিজিটাল ভেরিফিকেশন (Secure QR Verification)
প্রত্যেকটি প্যাথলজি রিপোর্ট একটি অনন্য ও নিরাপদ ক্রিপ্টোগ্রাফিক কিউআর-কোড ব্যাজ যুক্ত থাকে। ক্যামেরা দিয়ে স্ক্যান করা মাত্রই সেন্ট্রাল ক্লাউড প্ল্যাটফর্ম থেকে মূল স্বাক্ষরিত রিপোর্টটি অনলাইনে ভেরিফাই করা সম্ভব। এতে রিপোর্ট জালিয়াতি সম্পূর্ণভাবে রোধ হবে।

### ঙ. স্বয়ংক্রিয় গুগল ড্রাইভ ডেটা ব্যাকআপ ইঞ্জিন (Automated Google Drive Sync)
রোগীর সুবিধার্থে এবং গুরুত্বপূর্ণ মেডিকেল রেকর্ড সুরক্ষায় প্রতিটি অনুমোদিত ও রিভিউড রিপোর্ট সরাসরি ক্লিনিকের প্রাতিষ্ঠানিক গুগল ড্রাইভে এনক্রিপ্টেড ব্যাকআপ হিসেবে আপলোড হয়ে যায়।

### চ. চিকিৎসকদের জন্য টেলি-প্যাথলজি ডিজিটাল বোর্ড (Telepathology live Board)
রিয়েল-টাইম অডিও-ভিডিও সংযোগের মাধ্যমে দেশের বিভিন্ন প্রান্তে থাকা বিশেষজ্ঞ প্যাথলজিস্ট এবং কনসালট্যান্টরা যেকোনো রিপোর্টের জটিল বিষয়ে লাইভ ওয়েবআরটিসি (WebRTC) চেম্বারে বোর্ডে মিলিত হয়ে যৌথ সিদ্ধান্ত নিতে পারেন।

### ছ. ক্যান্সার ও মাল্টি-মোডাল প্রেডিক্টিভ এআই প্রাক-পরীক্ষক (Oncology Predictive AI Model)
মেডিকেল সায়েন্সের আধুনিকতম অগ্রগতি নিশ্চিত করতে আমরা আমাদের পোর্টালের সাথে একটি সহযোগী এআই প্রি-স্ক্যানার যুক্ত করেছি। এটি আপলোডকৃত ইমেজের পিক্সেল প্যাটার্ন ডিটেক্ট করে সম্ভাব্য টিউমার বা অনকোলজিক্যাল ঝুঁকি বিশ্লেষণের জন্য চিকিৎসকদের সতর্কবার্তা পাঠাতে পারে।

### জ. বারকোড দিয়ে সরাসরি স্পেসিমেন ও টিউব ট্র্যাকিং (NFC/Barcode Tracking Interface)
স্যাম্পল সংগ্রহের মুহূর্তেই প্রতিটি টিউবে বারকোড সাঁটিয়ে দেওয়া হয় এবং সিস্টেমে যুক্ত ক্যামেরার মাধ্যমে দ্রুত বারকোড স্ক্যানিং স্ক্রিন নির্দেশিকা (Alignment Guide) ব্যবহার করে তা ডাটাবেজের সাথে লিঙ্কড হয়ে যায়। কোন স্যাম্পল কখন কোন প্রসেসে আছে তা রোগীর স্বজনরা ট্র্যাকিং প্যানেল থেকে লাইভ ট্র্যাকিং আইডেন্টিফায়ারের মাধ্যমে জানতে পারবেন।

## ২. গিটহাব (GitHub) এবং ভার্সেল (Vercel)-এ অবিলম্বে লাইভ করার সম্পূর্ণ নির্দেশিকা

ভার্সেলে (Vercel) আপনার ডোমেইন **shahnazpathology.org** অলরেডি কানেক্টেড আছে। এই কোডটিকে সরাসরি লাইভ করার জন্য আপনার ভার্সেল টোকেন (`AQ.Ab8RN6Iqjl_CDOCyPgwo87sDl5qbX0HT_X3F7eNhBHkajK8yrA`) ব্যবহার করে সম্পূর্ণ গাইড নিচে দেওয়া হলো।

### আপনি যদি গিটহাব ব্যবহার করে অটোমেটিক ডিপ্লয় করতে চান:
আমরা গিটহাব অ্যাকশনস কনফিগারেশন (`.github/workflows/deploy.yml`) সিস্টেমে যুক্ত করে দিয়েছি। এটি সচল করতে নিচের ৩টি পদক্ষেপ নিন:

1. **গিটহাব রেপোজিটরি সেটিংস:** আপনার গিটহাব প্রজেক্টে যান এবং উপরের **Settings** ট্যাবে ক্লিক করুন।
2. **সিক্রেট অ্যাড করা:** বাম পাশের মেনু থেকে **Secrets and variables** -> **Actions** এ ক্লিক করুন। এরপর **New repository secret** বাটনে ক্লিক করুন।
3. **secrets অ্যাড করুন:**
   - **নাম দিন:** `VERCEL_TOKEN` 
   - **ভ্যালু দিন:** `AQ.Ab8RN6Iqjl_CDOCyPgwo87sDl5qbX0HT_X3F7eNhBHkajK8yrA`
   - একই উপায়ে আপনার ভার্সেল প্রজেক্ট থেকে অর্গানাইজেশন আইডি সংবলিত `VERCEL_ORG_ID` এবং প্রজেক্ট আইডি সংবলিত `VERCEL_PROJECT_ID` সিক্রেট হিসেবে যুক্ত করে দিন। 
   - এর পর গিটহাবে কোড পুশ (Push) করা মাত্রই সিস্টেম নিজে থেকে ভার্সেলে লাইভ বিল্ড আপলোড করে দেবে।

### আপনার গিটহাবের জন্য তৈরি করা কপি ও পেস্ট করার উপযোগী রেডিমেড প্রম্পট:

আপনি গিটহাবে সরাসরি কমান্ড ব্যবহারের জন্য নিচের প্রম্পটটি আপনার টার্মিনাল বা কম্যান্ড লাইনে রান করুন। এটি রান করার সাথে সাথে কোড গিটহাবের মাধ্যমে ভার্সেলের সাথে সিঙ্ক হয়ে লাইভ হয়ে যাবে:

```bash
# ১. গিট ইনিশিয়ালাইজ করুন ও ফাইল ট্র্যাকিং-এ যোগ করুন
git init
git add .
git commit -m "feat: integrate elite diagnostic modules & vercel deployment workflows"

# ২. আপনার গিটহাব রিমোট লিংক যুক্ত করুন (আপনার রেপো ইউআরএল দিয়ে পরিবর্তন করে নিন)
git branch -M main
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/shahnazpathology.git

# ৩. গিটহাবে কোড পুশ করুন
git push -u origin main
```

আপনার ভার্সেল অ্যাকাউন্টে প্রজেক্টের রুট ডিরেক্টরি হিসেবে `./` ডিফল্ট থাকবে এবং বিল্ড কম্যান্ড (Build Command) হিসেবে `npm run build` দিয়ে দেবেন। ভার্সেল আপনার ফ্রন্টএন্ড কোড সরাসরি বিল্ড করে আপনার নিজের ডোমেইন **shahnazpathology.org**-তে ইনস্ট্যান্ট লাইভ করে দেবে!
